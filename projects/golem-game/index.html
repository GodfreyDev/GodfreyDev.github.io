Golem Game

VR MMORPG focused on exploration and immersion

Golem Game is a VR MMORPG designed for a laid-back, exploratory experience across stand-alone VR, PC (with or without VR), and Android.

In Development VR PC Standalone Android

Overview

Players take on the role of wild golems, mana-based entities connected to the game world and its inhabitants. The core experience emphasizes exploration, interaction, and immersion, with combat as a supporting pillar rather than the primary focus.

XP Scaling and Progression

Scaling principle: use the Golden Ratio as the scaling value for XP requirements, aiming for a Phi increase every 5 levels.

XP formula (Python) 
import math

# Function to calculate XP required to reach level 'n' (0-indexed for formula)
# n=0 is for completing "level 0" and reaching Player Level 1.
def get_total_xp_for_formula_level(n, start_xp_for_n0=21, phi=1.618033988749895):
    if n < 0:
        raise ValueError("Formula level index 'n' must be a non-negative integer.")

    # Per-level multiplier derived from Phi/5 levels goal: phi ** (1/5)
    per_level_multiplier = phi ** (1.0 / 5.0)

    # XP for completing formula level 'n'
    xp = start_xp_for_n0 * (per_level_multiplier ** n)
    return int(round(xp))
